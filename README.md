# TinyFish Competitor Price Intelligence Agent

This project is built for the **TinyFish $2M Pre-Accelerator Hackathon**.

It is a **multi-product, multi-website price comparison application** that uses the **TinyFish Web Agent Automation API** to perform real-time, multi-step web automation across:

✅ Amazon  
✅ Flipkart  
✅ Reliance Digital  

The application extracts:

- Product title  
- Price  
- Rating  

Then prints a clean, human-readable comparison table and highlights the **cheapest seller**.

---

## ✅ Features

- Fully autonomous web agents (via TinyFish automation API)
- Multi-step workflows on 3 real Indian e-commerce websites
- Tracks 3 high-demand products:
  - iPhone 15 128GB Blue
  - Samsung S24 128GB Black
  - MacBook Air M2 13-inch 8GB/256GB
- Produces a clean CLI output report
- Sequential execution for stability
- Ideal for sellers, analysts, ecommerce teams
